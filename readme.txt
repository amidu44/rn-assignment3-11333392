# TaskManagerApp

TaskManagerApp is a simple task management application built with React Native. It includes a list of categories and a list of ongoing tasks, allowing users to add new tasks to the list.

## Features

- Display a list of categories such as Exercise, Study, Code, Cook, etc.
- Display a list of 15 ongoing tasks.
- Allow users to add new tasks.
- Custom components for categories and tasks.
